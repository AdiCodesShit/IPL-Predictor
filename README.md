# 🏏 IPL Match Winner Predictor

A Machine Learning project that predicts the outcome of Indian Premier League (IPL) matches and estimates win probabilities between two teams. Built using XGBoost and deployed with an interactive Streamlit web application.

---

## 🚀 Overview

This project explores how well a model can learn patterns from cricket match data — a domain known for its unpredictability.  

The model achieves ~55% accuracy, highlighting that match outcomes are influenced by many dynamic and uncertain factors, making them difficult to predict with high confidence.

---

## ✨ Features

- Predicts winner between two selected teams  
- Displays win probability (%) for both teams  
- Includes a confidence meter for predictions  
- Clean and interactive UI built using Streamlit  
- Modular and scalable ML pipeline  

---

## 🛠️ Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn  
- XGBoost  
- Streamlit  

---

## 📊 Model Details

- Algorithm: XGBoost Classifier  
- Task: Multi-class classification  
- Input Features:
  - Teams playing  
  - Toss winner  
  - Toss decision  
  - Match venue  
- Output:
  - Predicted winner  
  - Win probability (%)
